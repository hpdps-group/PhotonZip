# PhotonZip

This repository vendors [MANS](https://github.com/hpdps-group/MANS) as a git submodule under `./MANS`.

## Clone this repository

Recommended:

```bash
git clone --recurse-submodules <your-photonzip-repo-url>
cd PhotonZip
```

If you already cloned without submodules:

```bash
cd PhotonZip
git submodule update --init --recursive
```
